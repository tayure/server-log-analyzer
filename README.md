# Analisador-de-Logs-de-Servidor-e-Monitor-de-Seguran-a
Esta ferramenta em Python automatiza a análise técnica de logs de servidores. Utiliza Regex para extração de dados e o módulo ipaddress para validação e classificação de redes. Focada em performance e segurança, identifica tentativas de intrusão via blacklist e gera relatórios de saúde do servidor com baixo consumo de memória RAM.
